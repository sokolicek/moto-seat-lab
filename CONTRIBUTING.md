# Contributing

This repository is currently in concept and content architecture phase.

## Workflow

1. Read `AGENTS.md`.
2. Check `docs/STATUS.md`.
3. Keep changes small and documented.
4. Mark assumptions and research gaps.
5. Do not add code until the content/data concept is accepted.

## Content Rules

- Separate verified facts from assumptions.
- Add sources for technical motorcycle or seat data.
- Use confidence levels for uncertain data.
- Do not make medical claims.
- Include safety warnings for DIY seat modifications.
- Do not add affiliate links without disclosure.

## Before Publishing Changes

Check:

- concept still matches `docs/concept.md`,
- roadmap is updated if scope changes,
- data model docs are updated if new entity types are introduced.
