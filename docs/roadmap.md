# Roadmap

## Phase 0 - Concept

Status: In progress

- Define target audience.
- Define content architecture.
- Define first country and first motorcycle model.
- Define affiliate strategy.
- Define research backlog.

## Phase 1 - First Content MVP

Goal: create the first usable content set before building the full site.

Pages:

- Home concept.
- Why motorcycle seats hurt.
- Suzuki GSX-S1000GX comfort guide.
- DIY seat planning checklist.
- Materials and tools.
- Germany buying guide.

## Phase 2 - Website Prototype

Goal: build a simple static or lightweight web version.

Possible stack options:

- Astro
- Vite + static content
- Next.js only if needed later

Decision not made yet.

## Phase 3 - Monetization

Goal: add ethical affiliate monetization.

Potential programs:

- Amazon Associates,
- motorcycle accessory retailers,
- eBay Partner Network,
- country-specific shops,
- upholstery service referrals.

Must include affiliate disclosure.

## Phase 4 - Country Expansion

Planned countries:

- Germany,
- Austria,
- Switzerland,
- France,
- Italy,
- Slovakia,
- Hungary,
- Poland,
- Turkey,
- Thailand,
- Indonesia,
- Malaysia.

## Phase 5 - Bike Database Expansion

Start with:

- Suzuki GSX-S1000GX.

Then expand by:

- most common touring/adventure/sport-touring bikes in Germany,
- owner pain points,
- search demand,
- available seat products,
- affiliate availability.

## Phase 6 - Tools

Possible tools:

- comfort questionnaire,
- seat material recommendation wizard,
- budget estimator,
- DIY difficulty calculator,
- ride duration setup guide,
- printable workshop checklist.

## Phase 7 - Data And Calculation Engine

Goal: make the website capable of model-specific comparison and guided recommendations.

Planned:

- motorcycle database,
- stock seat database,
- aftermarket seat database,
- material database,
- configurable calculation rules,
- rider profile inputs,
- OEM vs aftermarket vs DIY comparison,
- confidence score and assumptions display.

## Phase 8 - Personal Seat Plan

Goal: let a rider generate a practical plan.

Output:

- recommended path,
- estimated cost,
- required materials,
- tools,
- risk warnings,
- testing checklist,
- affiliate product categories.
