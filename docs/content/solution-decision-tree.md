# Seat Comfort Solution Decision Tree

## Goal

Help riders choose the least risky useful solution first.

## Decision Tree

```text
Do you feel pain after less than 1 hour?
|
|-- Yes
|   |-- Is the pain mainly heat/sweat?
|   |   |-- Try 3D mesh cover or breathable riding pants.
|   |
|   |-- Is the pain sharp on sit bones?
|   |   |-- Try air cushion or evaluate foam support.
|   |
|   |-- Do you slide forward?
|   |   |-- Check seat slope, cover grip, posture, suspension sag.
|   |
|   |-- Is reach to ground already difficult?
|       |-- Avoid raising seat too much; consider reshaping, not only adding layers.
|
|-- No, only after long rides
    |-- Touring comfort problem.
    |-- Consider layered foam, professional upholstery, air cushion, or full comfort seat.
```

## Budget Decision

```text
Under 50 EUR
- 3D mesh
- simple cover
- posture and clothing check

50-150 EUR
- better pad
- air cushion
- gel pad
- basic materials

150-400 EUR
- professional foam modification
- better cover
- used spare seat for experiments

400+ EUR
- custom upholstery
- ready-made comfort seat
- heated custom seat
```

## Skill Decision

```text
No DIY skill
- buy add-on
- ask upholsterer

Basic DIY
- remove seat
- install cover/pad
- reversible experiments

Intermediate DIY
- foam shaping
- re-stapling
- gel insert

Advanced DIY
- full rebuild
- sewing cover
- heating integration
```

## Risk Ladder

Start with low-risk changes:

1. riding pants and posture,
2. 3D mesh,
3. air/gel pad,
4. spare used seat,
5. foam reshaping,
6. custom cover,
7. heating,
8. full rebuild.
