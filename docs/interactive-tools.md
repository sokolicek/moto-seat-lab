# Interactive DIY Tools Concept

## Goal

The future website should not be only an article library. It should also help riders calculate, compare, and plan seat comfort changes.

The tools should guide a rider from:

```text
my body + my motorcycle + my current seat problem
```

to:

```text
recommended seat solution paths, estimated cost, difficulty, risk, and comparison with OEM/aftermarket seats
```

## Proposed Tools

### 1. Rider Comfort Profile

Inputs:

- age,
- height,
- weight,
- inseam length,
- sit bone distance,
- hip width,
- riding experience,
- typical ride duration,
- pain start time,
- pain location,
- summer/winter riding,
- solo/passenger use,
- budget,
- DIY skill level.

Outputs:

- rider comfort profile,
- likely pressure risk zones,
- recommended first tests,
- warning if seat height/reach may be affected,
- suitable solution categories.

### 2. Motorcycle Seat Comparator

Inputs:

- motorcycle make/model/year,
- stock seat height,
- stock seat width,
- stock seat slope,
- rider triangle notes,
- aftermarket seat options,
- modified seat target.

Outputs:

- stock seat baseline,
- aftermarket seat comparison,
- DIY target comparison,
- expected seat height change,
- expected pressure distribution change,
- comfort/risk score.

### 3. DIY Seat Planner

Inputs:

- current seat dimensions,
- target comfort goal,
- selected material layers,
- layer thickness,
- material density/firmness,
- cover type,
- heating yes/no,
- waterproofing yes/no.

Outputs:

- layer stack plan,
- estimated final height,
- estimated cost,
- required tools,
- skill level warning,
- safety checklist,
- shopping list.

### 4. Seat Height Impact Estimator

Inputs:

- current seat height,
- rider height,
- inseam,
- planned added/removed material,
- boot sole height,
- motorcycle type.

Outputs:

- estimated new seat height,
- reach-to-ground risk,
- recommended maximum added height,
- warning if lowering may reduce foam support.

### 5. Ride Duration Recommendation

Inputs:

- pain start time,
- target ride duration,
- ride type,
- climate,
- rider weight,
- current seat type.

Outputs:

- minimum useful intervention,
- touring intervention,
- expected testing plan,
- product category suggestions.

### 6. Product And Material Matcher

Inputs:

- country,
- bike model,
- budget,
- problem type,
- DIY skill,
- climate.

Outputs:

- product categories,
- affiliate product slots,
- local retailer options,
- warning about unverified product fitment.

## Comparison Targets

The tools should support comparison between:

- OEM stock seat,
- OEM comfort seat,
- aftermarket seats,
- professional custom seats,
- DIY modified original seat,
- DIY modified second spare seat,
- add-on pads/covers.

Example aftermarket brands to research:

- Sargent,
- Corbin,
- Bagster,
- Touratech,
- Wunderlich,
- SW-Motech,
- Shad,
- Seat Concepts,
- local upholsterers.

Brand availability depends on country and motorcycle model.

## Important Warnings

Calculated results are estimates. The site should not promise exact medical or ergonomic outcomes.

The tools should show:

- confidence level,
- assumptions used,
- missing data,
- whether a value is measured, estimated, or user-entered.
