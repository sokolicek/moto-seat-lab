# Domain Decision

Status: preferred

Date: 2026-06-02

## Preferred Domain

```text
motoseatlab.com
```

## Reason

- short,
- memorable,
- matches the project name Moto Seat Lab,
- works internationally,
- avoids hyphens,
- suitable for SEO and brand use,
- can later support broader advisory content through subdirectories or domains.

## Defensive Domains To Consider

```text
motoseatlab.de
motoseatlab.eu
moto-seat-lab.com
```

## Caution

`motoseat.com` exists as a real motorcycle seat/cover business. Avoid names that look like the same shop or could confuse users.

Before buying, confirm availability with a registrar and do a basic trademark/conflict check.
