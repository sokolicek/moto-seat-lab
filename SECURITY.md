# Security Policy

## Scope

This is a concept-stage documentation repository for a future motorcycle seat comfort website and data tool.

It does not currently contain a running production application.

## Safety Notes

Motorcycle seat modifications can affect:

- rider control,
- reach to the ground,
- passenger safety,
- waterproofing,
- heating/electrical safety,
- warranty,
- resale value.

Content should not present DIY modifications as risk-free.

## Secrets

Do not commit:

- API keys,
- affiliate credentials,
- private tokens,
- passwords,
- private customer data,
- database exports.

If affiliate programs are added later, store credentials outside the repository.
